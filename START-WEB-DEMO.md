# 🌱 快速启动Web演示

## 一键启动

在项目根目录运行以下命令：

```bash
cd web-demo && npm install && npm run dev
```

浏览器会自动打开 `http://localhost:3001`

## 分步说明

### 1. 进入Web演示目录

```bash
cd web-demo
```

### 2. 安装依赖（首次运行需要）

```bash
npm install
```

这会安装以下依赖：
- React 18 - UI框架
- Recharts - 数据图表
- Vite - 开发服务器
- TypeScript - 类型支持

### 3. 启动开发服务器

```bash
npm run dev
```

你会看到类似这样的输出：

```
  VITE v5.0.0  ready in 500 ms

  ➜  Local:   http://localhost:3001/
  ➜  Network: use --host to expose
  ➜  press h to show help
```

### 4. 在浏览器中打开

浏览器会自动打开，如果没有，请手动访问：

**http://localhost:3001**

## 使用指南

### 界面说明

打开后你会看到：

1. **顶部标题栏** - 显示应用名称和标语
2. **实时状态卡片** - 显示LED状态指示灯和当前状态
3. **控制面板** - 三个操作按钮
4. **设备卡片** - 显示传感器实时数据
5. **数据趋势图** - 显示过去2小时的环境数据
6. **个性化建议** - 根据当前状态给出建议

### 操作步骤

#### 第一步：开始模拟

点击 **"▶ 开始模拟"** 按钮

- 系统会每3秒生成新的传感器数据
- LED指示灯会根据植物状态改变颜色
- 数据图表会实时更新

#### 第二步：观察状态

观察以下内容：

- **LED指示灯颜色**
  - 🟢 绿色 = 健康
  - 🟡 黄色 = 需要浇水
  - 🔴 红色 = 需要光照

- **传感器数据**
  - 💧 土壤湿度（目标 >30%）
  - ☀️ 光照强度（目标 >500 lux）
  - 🌡️ 温度
  - 💨 空气湿度

- **个性化建议**
  - 系统会根据数据给出具体建议

#### 第三步：执行操作

当系统提示需要照料时：

1. **需要浇水时**（湿度<30%）
   - 点击 **"💧 浇水"** 按钮
   - 观察湿度数值上升
   - LED可能变为绿色

2. **光照不足时**（<500 lux）
   - 点击 **"☀️ 移到光照处"** 按钮
   - 观察光照强度增加
   - LED可能变为绿色

#### 第四步：查看趋势

- 滚动到数据趋势图
- 观察各项指标的变化曲线
- 了解环境数据的历史趋势

## 常见问题

### Q: 页面打不开？

**A:** 检查端口3001是否被占用，可以修改 `vite.config.ts` 中的端口号。

### Q: 按钮点击没反应？

**A:** 确保先点击"开始模拟"按钮，某些按钮只在特定条件下可用。

### Q: 图表不显示？

**A:** 等待几秒让数据积累，或刷新页面重新开始。

### Q: 想要停止模拟？

**A:** 点击"⏸ 暂停模拟"按钮即可停止数据更新。

## 技术细节

### 数据模拟

- 每3秒生成一次新数据
- 数据范围符合实际传感器规格
- 包含随机波动模拟真实环境

### 状态判断逻辑

```
健康状态 = 湿度 > 30% AND 光照 > 500 lux
需要浇水 = 湿度 < 30%
需要光照 = 光照 < 500 lux
```

### 性能优化

- React.memo避免不必要的重渲染
- 数据限制在最近20个点
- 图表使用虚拟化渲染

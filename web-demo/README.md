# AI智能植物养护机器人 - Web演示版

这是一个可以在浏览器中运行的Web演示版本，展示了AI智能植物养护机器人的核心功能。

## 功能特性

- 🌱 **实时状态监测** - 显示植物的健康状态
- 📊 **数据可视化** - 环境数据趋势图表
- 💡 **智能建议** - 基于传感器数据的个性化建议
- 🎮 **交互模拟** - 模拟浇水、移动植物等操作
- 🔋 **电池监控** - 显示设备电池状态
- 🎨 **萌系界面** - 可爱友好的用户界面

## 快速开始

### 1. 安装依赖

```bash
cd web-demo
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将自动在浏览器中打开 `http://localhost:3001`

## 使用说明

### 主要功能

1. **开始模拟** - 点击"开始模拟"按钮，系统会每3秒生成新的传感器数据
2. **浇水操作** - 当植物需要浇水时（湿度<30%），点击"浇水"按钮
3. **移动到光照处** - 当光照不足时（<500lux），点击"移到光照处"按钮
4. **查看数据** - 实时查看传感器数据和趋势图表

### 状态指示灯

- 🟢 **绿色** - 植物健康，环境良好
- 🟡 **黄色** - 需要浇水
- 🔴 **红色** - 需要更多光照

### 传感器数据

- **土壤湿度** - 目标: >30%
- **光照强度** - 目标: >500 lux
- **温度** - 适宜: 20-28°C
- **空气湿度** - 适宜: 50-70%

## 技术栈

- **React 18** - UI框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Recharts** - 数据可视化
- **CSS3** - 样式和动画
